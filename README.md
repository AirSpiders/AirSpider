## Overview👀

> + **AirSpider** is a high-performance asynchronous crawler framework for developers 🚀
> + Based on Redis: task distribution, task deduplication, and distributed ☁️
> 

## Requirements☁️

- Python 3.6➕
- Works on Linux, Windows, macOS🍎

## Features🌲

+ **Quick to Start ☑️**
+ **Low Coupling ☑️**
+ **High Cohesion ☑️**
+ **Easy Expansion ☑️**
+ **Orderly Workflow ☑️**

## Installation🔨

``` shell
---------------------------

# For Linux && MacOS🔥
pip3 install airspider

---------------------------

# For Windows🔥
pip3 install airspider
```
